# Chengeta AI — Brand Guidelines

> **Chengeta** — _chiShona_ verb, **ku-chengeta**: *to store, to preserve, to protect, to keep safe.*

These guidelines define how the Chengeta AI identity is used across the README, documentation,
website, terminal, and social surfaces. The goal is a brand that reads as **African, modern,
enterprise-grade, and developer-first** — never tribal-kitsch, safari-themed, or cartoonish.

---

## 1. The Name

- **Product name:** Chengeta AI
- **Package / import:** `chengeta-ai` (PyPI) → `import chengeta_ai`
- **CLI command:** `chengeta`
- **Pronunciation:** _chen-GET-ah_ (hard *g*)
- **Origin:** chiShona (Zimbabwe), the verb *to preserve / keep safe* — the essence of a memory layer.

Always write the display name as **Chengeta AI** (two words, capital C, capital AI). Never
`chengeta ai`, `ChengetaAI`, or `CHENGETA AI` in prose. The lowercase `chengeta-ai` form is
reserved for the package name and code contexts.

### Tagline

> **Persistent Memory for Intelligent Agents**

Approved alternates (use sparingly, never invent new ones):

- Memory Infrastructure for Agentic AI.
- Cache Less. Think More.
- Smart Memory. Faster Agents.
- Built for AI Systems That Remember.

### The mark

The mark is a **memory-vault infinity** — two interlocking loops forming an infinity figure,
crossed by a gold vault core with a keyhole. It symbolises:

- **∞ infinity** — persistence and continuous learning
- **interlocking loops** — memory that links across turns and sessions
- **gold vault core** — what is stored is *kept safe* (chengeta)

### Clear space & sizing

- Maintain clear space around the lockup equal to the height of the mark's gold core on all sides.
- Minimum mark size: **24 px**. Minimum lockup width: **120 px**.
- Never stretch, recolour outside the palette, rotate, add drop shadows, or place the
  full-colour lockup on a busy photo. Use the reversed or monochrome variant instead.

## 3. Colour System

### Primary

| Token | Hex | Role |
|-------|-----|------|
| Forest Green | `#0F5B3A` | Primary brand colour, headings, mark right loop |
| Gold | `#C9A227` | Accent, vault core, emphasis, calls-to-action |
| Black | `#111111` | Text on light, keyhole |
| White | `#FFFFFF` | Surfaces, text on dark |

### Secondary

| Token | Hex | Role |
|-------|-----|------|
| Deep Emerald | `#1B7F5A` | Secondary brand, mark left loop, links |
| Bright Emerald | `#3DBF86` | Interactive / hover, on-dark accents |
| Light Sand | `#F7F3E8` | Warm neutral surface, callouts |

### Dark-theme surfaces (web + docs)

| Token | Hex |
|-------|-----|
| Background | `#0B140F` |
| Panel | `#10211A` |
| Border | `#1C3B2C` |
| Muted text | `#9DB3A6` |
| Body text | `#EAF2EC` |

### Signature gradient

`linear-gradient(135deg, #3DBF86, #C9A227)` — emerald → gold. Used for the wordmark accent,
primary buttons, and stat figures. Use it intentionally and never as a full-page wash.

> **Contrast:** Body text (`#EAF2EC`) on background (`#0B140F`) and Forest Green (`#0F5B3A`) on
> White both exceed WCAG AA 4.5:1. Gold (`#C9A227`) is an *accent* — pair it with black text, not
> as small text on white.

## 4. Typography

| Role | Typeface | Notes |
|------|----------|-------|
| Display / headings | **Plus Jakarta Sans** (800/700) | Geometric, modern, tight tracking (`-0.02em`) |
| Body / UI | **Inter** (400/500/600) | Workhorse text |
| Code / terminal | **JetBrains Mono** / Fira Code | Monospace |

Fallback stack: `'Plus Jakarta Sans', 'Inter', 'Segoe UI', -apple-system, system-ui, sans-serif`.

Type scale (web): `0.78` · `0.875` · `1` · `1.18` · `1.5` · `2.2` · `4.2 rem`, line-height `1.1`
for display and `1.6–1.7` for body.

---

## 5. Visual Texture

Use **geometric, African-inspired** motifs — chevron bands, triangle notches, interlocking
diamonds — rendered cleanly in gold or emerald at low opacity. These echo Shona stonework and
textile geometry without resorting to literal or stereotyped imagery.

**Avoid:** tribal masks, safari/animal motifs, mud-hut clip art, hand-drawn or cartoon styles,
and "purple-gradient AI" tropes.

---

## 6. Voice & Tone

Professional, modern, developer-focused. Confident but not hyperbolic. We talk about **memory**,
**persistence**, **recall**, and **keeping work safe** — the language of the name.

- ✅ "Chengeta AI gives agents a persistent memory layer."
- ✅ "Work done once is never paid for twice."
- ❌ "Revolutionary AI-powered turbo cache!!!"
